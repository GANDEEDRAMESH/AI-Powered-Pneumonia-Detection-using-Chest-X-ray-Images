# AI-Powered Pneumonia Detection using Chest X-rays

## Overview
This project focuses on building a deep learning pipeline to detect pneumonia from chest X-ray images.

## Features
- CNN, ResNet, MobileNet models
- Hyperparameter tuning (Random Search)
- Evaluation using Precision, Recall, F1-score
- Error Analysis on misclassified images
- Grad-CAM for explainability
- Streamlit deployment

## Tech Stack
Python, PyTorch, OpenCV, Scikit-learn, Streamlit
